# CabeleleilaLeila - README

## Configuração do repositorio

#### Frontend - Next.js
- node version v21.6.0
- npm install
- npm run dev

#### Backend - Node
dotnet version 8.0.405

Instalar Dependencias
- dotnet restore

Compilar o projeto
- dotnet build

Aplicat Migrations
- dotnet ef database update

Executar o projeto
- dotnet run
